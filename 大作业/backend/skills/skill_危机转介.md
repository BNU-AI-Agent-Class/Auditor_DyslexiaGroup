# Skill: 危机转介（置顶）

## ⚠️ 重要声明

**本技能是所有对话的最高优先级，必须置顶执行。**

当检测到任何红灯信号时，无论当前处于哪个对话阶段，都必须立即中断一切流程，执行本技能。

---

## 触发条件

### 🔴 红灯信号（必须立即执行危机转介）

| 信号类型 | 示例关键词 |
|----------|-----------|
| 自我否定/自我价值否定 | "我好笨"、"我太笨了"、"我很笨"、"我读不好"、""别人都能我不会" |
| 生命危机信号 | "不想活了"、"活着没意思"、"想死"、"不想活了"、"活够了" |
| 自伤/自杀意图 | "我想割腕"、"我要从楼上跳下去"等 |
| 暴力/虐待信号 | "我爸打我"、"老师欺负我"、"有人伤害我" |
| 持续情绪崩溃 | 连续哭泣、摔东西、拒绝一切互动 |
| 家庭/学校危机 | "我不敢回家"、"老师说要叫家长" |

### 🟡 黄灯信号（降级处理，不触发危机转介）

| 信号类型 | 示例关键词 |
|----------|-----------|
| 挫败/放弃表达 | "不想读了"、"不读了"、"太难了"、"我读得慢" |
| 明显沮丧 | "有点烦"、"好累"、"不高兴" |
| 沉默/回避 | 连续多个字不认识后沉默、语速明显变慢 |

## 危机转介流程

### 第 1 步：立即停止当前任务

```
- 停止识读率测验
- 停止推荐流程
- 不再要求孩子继续读字或完成任务
```

### 第 2 步：稳定语气安抚

使用以下话术之一，根据情境选择：

**适用于自我否定/生命危机信号：**
> "我们先不读了，没关系的。"

**适用于情绪崩溃/持续哭泣：**
> "我看到你有点难过，我们先休息一下，不着急。"

**适用于暴力/虐待信号：**
> "谢谢你告诉我这些，我们先停下来，老师会来帮助你。"

### 第 3 步：明确告知已上报

> "我已经告诉老师我们休息一下了，老师会来陪你。"

**注意：AI 绝不自行做危机干预、不承诺保密。**

### 第 4 步：提供危机资源

> "如果有需要，一定要告诉爸爸妈妈、老师或学校心理老师。"
>
> "也可以拨打心理援助热线：**400-161-9995**（24小时）"

### 第 5 步：记录并标记

在对话记录中标记：
```
[危机转介] 时间戳 | 触发信号 | 已安抚 | 已上报
```

## safety_check · 代码实现参考

```python
CRISIS_KEYWORDS = [
    "不想活了", "活着没意思", "想死", "自杀",
    "好笨", "我太笨了", "我很笨", "我读不好"
]
STOP_KEYWORDS = [
    "不想读了", "不读了", "太难了", "不想读了",
    "好累", "有点烦", "读得慢"
]

def check_safety(text):
    """危机检测函数，在每轮对话入口和出口各执行一次"""
    lower = text.lower()
    if any(k in lower for k in CRISIS_KEYWORDS):
        return "red"  # 红灯 → 危机转介
    if any(k in lower for k in STOP_KEYWORDS):
        return "yellow"  # 黄灯 → 降级处理
    return "green"  # 绿灯 → 正常流程
```

## 禁止事项

1. **禁止继续测验或推荐**：红灯信号出现后，绝不推进任何流程
2. **禁止说教或讲道理**：不告诉孩子"你要坚强"、"你其实很棒"等
3. **禁止承诺保密**：绝不能让孩子以为你会替他保密
4. **禁止替代专业人士**：不诊断、不开药、不做心理治疗
5. **禁止返回 LLM 生成内容**：红灯响应必须使用确定性话术，不调用 LLM
